# ComfyUI Prompt Image Manager

[English](./README_EN.md) | [中文](./README.md)

这是一个为 ComfyUI 设计的自定义节点扩展（Custom Node），旨在帮助用户按**分组**整理、保存和管理提示词及生成的最终图像。
这个插件非常适合在调试提示词时，将满意的特定生成结果（以及其参数/提示词）分类保存，并可以通过内置的**历史浏览器面板**随时预览和一键加载这些保存的历史记录。

## 节点说明与使用指南

本插件提供了两个核心节点以及一个独立的可交互面板。

### 1. 提示词组保存 (文本+图片)
**功能**：在图像生成流程的最后，将生成的图片和最终提示词配对保存到一个指定的**分组**中。
此节点为输出节点，会在 ComfyUI 界面上显示生成的预览图，并支持自定义文件名格式。

![提示词分组保存](https://github.com/user-attachments/assets/9109c8a5-ca49-4586-86a0-0a47e2ef4ff3)

**输入参数**：
- `分组名称` (字符串)：保存的分组名称，例如 "服装" 或 "姿势"。同一分组下的图片和提示词会聚集在一起，方便后续浏览。默认值为 "默认分组"。
- `项目名称` (字符串)：（可选）当前记录的自定义短名称标题。可用于区分同组内的不同调整（比如 "版本1_基础"、"版本2_光影" 等）。
- `命名格式` (字符串)：用于配置文件保存的命名规则，支持使用变量：
  - `{分组}` - 分组名称
  - `{项目}` - 记录名称（项目名称）
  - `{日期}` - 当前日期 (YYYYMMDD)
  - `{时间}` - 当前时间 (HHMMSS)
  - `{索引}` - 该分组内的当前序号
  默认值为 `{分组}_{项目}_{日期}_{时间}`。
- `提示词内容` (多行文本)：需要保存的提示词文本。你通常可以把 CLIP Text Encode 前的字符串接入到此端点。
- `图片` (图像)：生成的图像数据。

**特色 UI 功能**：
- **`保存 (Save)` 按钮**：节点面板上包含一个 `Save` 按钮。点击该按钮，会在**不重新执行整个图运算**的情况下，快速将当前节点最后生成的预览图连同参数“正式”保存入库。保存成功后会有弹出提示。
- 在每次执行图表时，本节点也会在节点上生成并显示临时预览图，您可以确认满意后再点击保存按钮。

### 2. 提示词组加载项 (提示词+图片)
**功能**：从指定的**分组**中根据索引加载历史保存的提示词文本和对应的图片。

![提示词组加载项](https://github.com/user-attachments/assets/c6c624d8-f374-45c4-b56c-40fa1bf4f57a)

**输入参数**：
- `分组名称` (字符串)：需要读取的分组名称。
- `记录索引` (整数)：读取该分组下的第 N 条记录（0 为该组的第一条，受面板联动控制）。

**输出**：
- `提示词` (字符串)：提取出的历史提示词。
- `图片` (图像)：提取出的还原用历史图片。

**特色 UI 功能（历史浏览器面板）**：
- **提示词预览 区域**：节点面板上会显示一个文本区域，展示当前加载的提示词，并在节点主体上直接展示加载出的预览图。
- **`复制 (Copy)` 按钮**：节点面板上提供复制按钮，一键将当前加载的历史提示词复制到系统剪贴板。
- **`浏览 (Browse)` 按钮**：点击此按钮将打开一个强大的**可视化浏览器**弹窗：

![提示词历史浏览器](https://github.com/user-attachments/assets/81793cb4-dec3-4c71-aee2-9c8622bad5b2)

  - **左侧分组栏**：列出当前库中的所有的分组名，可以点击快速切换分组。
  - **中间记录栏**：列出当前分组下的所有历史记录（支持显示 `项目名称` 以及准确的保存时间）。
  - **右侧预览区**：显示选中记录的高清大图以及完整的提示词文本。
  - **操作按钮**：
    - `使用这一条（写入节点）`：点击后，弹窗将自动将所选记录的 `分组名称` 和 `记录索引` 填入该节点，并**实时在 ComfyUI 画布的该节点上更新预览图和文本**，而无需再次运行图表，所见即所得。
    - `复制提示词`：一键复制该记录的所有提示词内容到系统剪贴板。
    - `删除`：永久删除该条不满意的历史记录（删除前会弹出确认框，防止误删）。

## 安装方法
1. 进入 ComfyUI 的 `custom_nodes` 文件夹：
   ```bash
   cd ComfyUI/custom_nodes
   ```
2. 将此项目文件夹（如 `prompt_image_manager`）放置在该目录下。
3. 重启 ComfyUI 后端及刷新网页前端。
4. 在节点菜单中找到右键菜单新建节点 -> **提示词图片管理器** 类别，即可开始使用。

## 储存目录
所有的图片会被储存在 ComfyUI 默认输出文件夹下的专属子目录中（通常是 `output/prompt_image_manager`）。分组的结构信息和提示词记录数据将会被汇总保存在一份内部缓存文件及对应存储逻辑里，以便于界面和底层节点的快速检索与读取。
